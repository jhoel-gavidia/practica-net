# practica-net
Sistema CRUD - ASP.NET Core MVC

Este proyecto es una aplicación web desarrollada ASP.NET core MVC que implementa
un sistema CRUD para la gestión de colaboradores y empresas.

## Tecnologías utilizadas
- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server (o el motor que uses)
- HTML, CSS, JavaScript
- Bootstrap

## Arquitectura del proyecto
- Controllers → lógica de negocio
- Models → entidades y datos
- Views → interfaz de usuario
- Data → acceso a base de datos y migraciones
